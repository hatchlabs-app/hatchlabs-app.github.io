# hatchlabs-web

Static site for Hatch Labs, intended for GitHub Pages.

## Structure

- `/docs/` public Hatch Labs site for GitHub Pages
- `/drafts/hidey/` non-public draft for the Hidey landing page
- `/privacy/`, `/legal/`, `/index.html`, `/styles.css` local working copies

The Hidey landing page is currently kept out of the public publish path while Hatch Labs launches first.

## Publish on GitHub Pages

1. Create a new GitHub repository named `hatchlabs-web`.
2. Push these files to the default branch.
3. In GitHub repository settings, enable Pages from the `main` branch and `/docs` folder.

If you want to preview locally, open `index.html` in a browser or run a simple static server.
